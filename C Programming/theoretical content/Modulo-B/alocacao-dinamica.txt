Todo estudo anterior foi baseado em alocação de memória estática do PC, ou seja, espaço definido antes da execução do programa. 

1. Alocação Dinâmica

    Pode ser solicitado espaços de memória durante a execução do programa, o que é conhecida como alocação dinâmica.

    -> Biblioteca utilizada: <stdlib.h> -- Funções principais: malloc e free

    -> Função Malloc

        Declaração: void* malloc(size_t size)  

        Funcão espera receber uma qtd. de bytes para alocação, se alocar a quantidade, a função retorna a posição de memória referente a primeira posição alocada, caso contrário a função retorna um ponteiro nulo.

            malloc(4) // aloca 4 bytes
        
        Para utilizar a memória solicitada, precisa de um ponteiro.

            int *p;
            p = malloc(4) // p aponta para a primeira posição

        DEfinição de tamanho do Malloc:

            p = malloc(sizeof(int))   // aloca espaço de um inteiro (4B)
            p = malloc(sizeof(int)*2) // aloca espaço de dois inteiros (8B)
            p = malloc(sizeof(char)*10) // aloca espaço de 10 caracteres (10B)
            p = malloc(sizeof(int)*n) // aloca espaço de n inteiros ('n'B)

        Retorno do Malloc:

           A função retorna a primeira posição de memória. 
           Foi definido para essa função retornar um ponteiro para void (void*).
           Gera erro de compilação ao tentar atribuir um ponteiro para void para outro ponteiro, sendo necessário utilizar o cast de dados.

           Cast de Dados: simboliza ao pc converter um tipo de dado em outro.

                p = (int*)malloc(sizeof(int)*5) // o que esta dentro de malloc será convertido para o tipo definido antes.
        
    -> Função Free

        Declaração: void free(void *ptr)

        Para quando ñ quiser mais utilizar determinada posição de memória, a função vai desalocar toda a região alocada dinamicamente que inicia na posição fornecida.

            int *p = (int*) malloc(sizeof(int)*10); // ponteiro pra int para receber 10 inteiros
            free(p); // para quando ñ quiser mais utilizar esses 10 inteiros
        
    -> Função Calloc

        Mesma coisa do Malloc, mas solicita um número específico de itens de um determinado tamanho em bytes.

        Declaração: void* calloc(size_t num, size_t size);

            calloc(4, sizeof(int));
            // Aloca memória para 4 itens que ocupam cada um o espaço de um inteiro.

        Os itens especificados serão preenchidos com o valor 0 e retornará um ponteiro para a primeira posição de memória

        A função free pode ser utilizada para liberar a memória da mesma forma que no malloc.

        Recomenda-se a utilização de calloc para a criação de vetores por ser mais intuitivo e facilitar o entendimento do código.

            int *p = calloc(9, sizeof(int));
            // Cria um vetor de 9 posições de 4 bytes cada uma, todas preenchidas com 0.
