---= Modularização e Ponteiros =---

1. Modularização

    Significa dividir o programa em módulos para que cada módulo execute uma tarefa específica.
    Até agora tudo foi executado em um único módulo, a função 'main', ou função principal.

    Isso fragmenta o programa em partes pequenas para realizar as tarefaz uma de cada vez:

      - Facilita o debug;
      - Melhora a legibilidade;
      - Elimina redundâncias.

 --> Função: é como um programa é modularizado, dividindo-o em várias funções que recebe determinados parâmetros, realiza o procedimento deles e retornar algum valor. (f(x) = 2x)

    -> Especificações para Funções em C

        - Tipo de retorno; (void = não retorna nada)
        - Nome da função;
        - Parâmetros; (tipos identificados e se +1, separados por vírgulas)
        - Código.

        <tipo de retorno> <nome> (parâmetro){
            <código>
        }

        void imprime(){ // parâmetro vazio e nome "imprime"
            printf("Neps Academy\n");
        } 
        int main(){
            imprime(); // pode ser chamado +1 vez.
        }

    -> Tipos de Funções

        Int - retorna valores inteiros como de cálculos de soma com inteiros;
        FLoat - retorna valores reais como de cálculos de médias aritméticas;
        Void - não retorna valores, é usado para exibir mensagens e menus, que não precisa de parâmetros.
               como não tem resultado a ser armazenado, não é utilizado nenhum return.

        2.0 x 2 - com ponto flutuante é utilizado em funções do tipo float senão toda a expressão é considerada como inteira e pode apresentar erros, mas é possível realizar um type casting e converter explicitamente um valor para outro tipo.

           ex.: (a + b)/2 considere que a e b são do tipo float, então deveria utilizar 2.0
                convertendo: return (float)(a + b)/2

2. Ponteiros

    Variável especial que armazena o endereço de outra variável na memória. Ao invés de armazenar o valor atribuido, ele armazena o endereço de memória onde esse valor está guardado.

    Imagine uma rua cheia de casas numeradas onde cada variável fica armazenada dentro das casas (endereços), o ponteiro funciona como um mapa que mostra em qual casa está guardada a informação que necessitamos acessar.

    Template de um ponteiro: <tipo> *<nome>;

    Aritmética: soma/subtração de valores de um ponteiro, o endereço ao qual ele aponta será alterado. Se somar 1 a um ponteiro que aponta para um int, então ele passará a apontar para a próxima região de memória.

        int *p;
        p + 1 -> ñ significa 1B a mais e sim próximo int na memória.

        Se um int ocupa 4B, então p + 1 aponta para um endereço 4B à frente de p.
    
    Tamanho de tipos: C possui um operador unário que retorna o tamanho em bytes de algo, como a quantidade de memória ocupada por um dado específico. 
    
        sizeof;

        sizeof(int) = 4
        sizeof(float) = 4
        sizeof(double) = 8
        sizeof(char) = 1 

        Os valores indicam que, por exemplo, o inteiro ocupa 4B de memória ao ser armazenado, se adicionar 1 ao ponteiro, então o novo endereço de memória apontado por ele estará 4B à frente do antigo.

    Alterando o valor do ponteiro: aqui é atribuído um endereço de memória à um ponteiro, assim como é atribuido valores ára variáveis comuns.

        Operador &; - descobrir o endereço de um var na memória e atribuí-lo aoa ponteiro.

        int x = 5;
        int *p = &x; - cria o ponteiro int p e atribui o endereço de x a ela;
        int y = 9;
        p = &y; agora p aponta para y.

    Acessando o conteúdo apontado por um ponteiro: usando o printf()

        int x = 5;
        int *p = &x;
        printf("%d\n", p);

        Saída: 6422296. 
        
        Esse valor de saída é referente ao endereço de memória a qual está localizado a variável x. Esse valor é unitário e diferente para cada pc e máquina que rodar o mesmo código.

        para saber o valor da variável x pelo ponteiro: 

            printf("%d\n", *p);

        É acessado indiretamente a variável x.

    Outras possibilidades:

        - Atribuir um endereço de um ponteiro a outro;
        - Atribuir o valor da variável apontada por um ponteiro a outra variável.

           int var = 123;
           int *ponteiro = &var;
           printf("%d\n", *ponteiro);
           *ponteiro = 789;
           printf("%d\n", var);

        Saída: 123 / 789.

        - operação aritmética:

           printf("Tamanho de um inteiro: %d\n\n", sizeof(int));

           int x = 1;
           int *p = &x;

           printf("p aponta para %d\n", p);
           printf("p+1 aponta para %d\n", p+1);
           printf("p+2 aponta para %d\n", p+2);

           Saída:

              Tamanho de um inteiro: 4

              p aponta para 6422296
              p+1 aponta para 6422300
              p+2 aponta para 6422304

            Ao adcionar uma unidade ao endereço armazenado, o descolacamento da posição da memória foi exatamente de 4B, nas duas vezes.
